# Solo Leveling Shadows Lightboxes
MMED 1054 assignment focused on using lightboxes with CSS. The page features a series of characters from the tv show / anime Solo Leveling that when interacted with open up into lightboxes that include brief descriptions and facts about the characters.

## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open index.html in the browser of your choice.
